## 📋 部署步骤（超详细）

### 第1步：准备GitHub账号

1. 访问 https://github.com
2. 如果没有账号，点击"Sign up"注册（免费）
3. 如果有账号，直接登录

### 第2步：上传代码到GitHub

#### 方法A：使用Git命令（推荐）

在你的项目文件夹打开终端，执行：

```bash
# 1. 初始化Git仓库
cd "d:\系统默认桌面存储\vscode_work\ai-personality-test"
git init

# 2. 添加所有文件
git add .

# 3. 创建第一次提交
git commit -m "AI人格测试系统 v1.0"

# 4. 在GitHub创建新仓库
# 访问 https://github.com/new
# 仓库名填：ai-personality-test
# 设置为Public（公开）
# 不要勾选任何初始化选项
# 点击"Create repository"

# 5. 关联远程仓库（替换YOUR_USERNAME为你的GitHub用户名）
git remote add origin https://github.com/YOUR_USERNAME/ai-personality-test.git

# 6. 推送代码
git branch -M main
git push -u origin main
```

#### 方法B：使用GitHub网页上传

1. 访问 https://github.com/new
2. 仓库名填：`ai-personality-test`
3. 设置为Public（公开）
4. 点击"Create repository"
5. 点击"uploading an existing file"
6. 把整个`ai-personality-test`文件夹拖进去
7. 点击"Commit changes"

### 第3步：部署到Vercel

1. **访问Vercel**
   - 打开 https://vercel.com
   - 点击"Sign Up"注册（用GitHub账号登录最方便）

2. **导入项目**
   - 登录后点击"Add New..." → "Project"
   - 选择"Import Git Repository"
   - 找到你的`ai-personality-test`仓库，点击"Import"

3. **配置项目**
   ```
   Project Name: ai-personality-test（可以改成你喜欢的名字）
   Framework Preset: Next.js
   Root Directory: frontend  ⚠️ 重要！必须设置为frontend
   Build Command: npm run build
   Output Directory: .next
   Install Command: npm install
   ```

4. **部署**
   - 点击"Deploy"按钮
   - 等待2-3分钟（会显示构建进度）
   - 看到"🎉 Congratulations!"就成功了

5. **获取链接**
   - 部署成功后会显示你的网站链接
   - 格式：`https://ai-personality-test-xxx.vercel.app`
   - 这就是你要分享给网友的链接！

## 🔧 常见问题

### Q1: 部署后链接打不开？
A: 等待2-3分钟，Vercel需要时间构建。刷新页面试试。

### Q2: 可以自定义域名吗？
A: 可以！在Vercel项目设置里添加自己的域名（需要购买域名）

### Q3: 免费版有限制吗？
A: Vercel免费版每月100GB流量，个人使用完全够用

### Q4: 如何更新内容？
A: 修改代码后推送到GitHub，Vercel会自动重新部署

### Q5: 链接太长怎么办？
A: 可以用短链接服务（如 suo.im）生成短链接
