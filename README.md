# AI时代人格测试系统 🤖✨

一个结合MBTI人格测试、AI适应力评估和工作环境偏好分析的现代化测试系统。

## 🌟 特色功能

### 1. 炫酷星座动画首页
- 12星座图标圆形排列
- 浮动动画效果
- 赛博朋克风格设计
- 星空背景特效

### 2. 60道科学题库
- **人格测试（30题）**：基于MBTI理论，评估E/I、S/N、T/F、J/P四个维度
- **AI适应力（20题）**：评估对AI工具的接受度、学习能力和应用意愿
- **工作环境（10题）**：评估远程工作、团队协作、创新倾向等偏好

### 3. 详细个性化报告
- MBTI人格类型分析（16种人格）
- AI适应力评分（0-100分）
- 工作环境偏好分析
- 职业方向推荐
- 综合分析和建议

### 4. 现代化UI设计
- 赛博朋克风格
- 霓虹色彩和渐变效果
- 流畅的动画过渡
- 完全响应式设计（支持手机和桌面）

## 🚀 快速开始

### 环境要求
- Node.js 18+
- npm 或 yarn

### 安装步骤

1. 克隆项目
```bash
cd ai-personality-test/frontend
```

2. 安装依赖
```bash
npm install --cache .npm-cache
```

3. 启动开发服务器
```bash
npm run dev
```

4. 打开浏览器访问
```
http://localhost:3000
```

## 📱 使用流程

1. **首页**：欣赏炫酷的星座动画，点击"开始测试"
2. **数据采集**：输入性别、生日、选择星座
3. **答题**：完成60道题目（约5-8分钟）
4. **查看报告**：获得详细的个性化分析报告

## 🎨 技术栈

- **前端框架**：Next.js 14 (App Router)
- **开发语言**：TypeScript
- **样式方案**：Tailwind CSS
- **状态管理**：React Hooks + LocalStorage
- **部署平台**：Vercel（推荐）

## 📊 MBTI人格类型

系统支持16种MBTI人格类型：

| 分析师 | 外交家 | 守护者 | 探险家 |
|--------|--------|--------|--------|
| INTJ | INFJ | ISTJ | ISTP |
| INTP | INFP | ISFJ | ISFP |
| ENTJ | ENFJ | ESTJ | ESTP |
| ENTP | ENFP | ESFJ | ESFP |

## 🤖 AI适应力等级

- **AI先锋（80-100分）**：积极拥抱AI，善于利用AI工具
- **AI探索者（60-79分）**：对AI持开放态度，正在学习中
- **AI观望者（40-59分）**：对AI有兴趣但缺乏实践
- **AI新手（0-39分）**：刚开始接触AI，需要更多学习

## 🎯 核心算法

### 人格计算
根据答题选项累计E/I、S/N、T/F、J/P四个维度的分数，最终确定MBTI类型。

### AI适应力计算
基于20道AI相关题目的得分，计算0-100分的AI适应力指数。

### 工作偏好分析
- 远程工作倾向：基于独立性和自主性相关题目
- 团队协作倾向：基于E/I维度和协作相关题目
- 创新倾向：基于N维度和创新相关题目

## 🚀 部署指南

### Vercel部署（推荐）

1. 注册Vercel账号：https://vercel.com
2. 连接GitHub仓库
3. 选择`frontend`目录作为根目录
4. 点击Deploy

### 本地构建

```bash
npm run build
npm start
```
