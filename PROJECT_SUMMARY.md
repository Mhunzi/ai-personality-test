## 📈 核心算法

### MBTI计算
```typescript
// 累计四个维度的分数
E/I: 外向/内向
S/N: 实感/直觉
T/F: 思考/情感
J/P: 判断/感知

// 最终确定16种人格类型之一
```

### AI适应力计算
```typescript
// 基于20道AI相关题目
总分 = Σ(每题得分)
百分比 = (总分 / 最高分) × 100

// 分级
80-100: AI先锋
60-79:  AI探索者
40-59:  AI观望者
0-39:   AI新手
```

### 工作偏好计算
```typescript
远程工作倾向 = f(I维度, 独立性题目)
团队协作倾向 = f(E维度, 协作题目)
创新倾向 = f(N维度, 创新题目)
```
