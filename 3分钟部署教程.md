# 🎯 3分钟部署教程（小白版）

## 你现在的情况

✅ 项目已经开发完成  
✅ 本地可以运行（http://localhost:3001）  
❌ 但只有你自己能访问，网友打不开  

## 目标

🎯 获得一个网址，比如：`https://ai-test.vercel.app`  
🎯 网友点击就能玩，不用下载安装  

---

## 方法1：最简单（推荐新手）

### 第1步：注册Vercel（1分钟）

1. 打开 https://vercel.com
2. 点击右上角 "Sign Up"
3. 选择 "Continue with GitHub"（用GitHub登录）
4. 如果没有GitHub账号，会自动跳转注册（免费）
5. 授权Vercel访问你的GitHub

### 第2步：上传代码到GitHub（2分钟）

**方式A：用GitHub Desktop（最简单）**

1. 下载GitHub Desktop：https://desktop.github.com
2. 安装并登录
3. 点击 "File" → "Add Local Repository"
4. 选择你的项目文件夹：`d:\系统默认桌面存储\vscode_work\ai-personality-test`
5. 点击 "Publish repository"
6. 仓库名填：`ai-personality-test`
7. 取消勾选 "Keep this code private"（设为公开）
8. 点击 "Publish repository"

**方式B：用命令行**

在项目文件夹右键 → "Git Bash Here"，输入：

```bash
git init
git add .
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/你的用户名/ai-personality-test.git
git push -u origin main
```

### 第3步：在Vercel部署（1分钟）

1. 回到 https://vercel.com
2. 点击 "Add New..." → "Project"
3. 找到 `ai-personality-test`，点击 "Import"
4. **⚠️ 重要：** 点击 "Root Directory" 右边的 "Edit"
5. 选择 `frontend` 文件夹
6. 点击 "Continue"
7. 点击 "Deploy"
8. 等待2-3分钟...
9. 看到 "🎉 Congratulations!" 就成功了！

### 第4步：获取链接

部署成功后，页面会显示你的网站地址，比如：
```
https://ai-personality-test-abc123.vercel.app
```

**这就是你要分享的链接！** 复制它，发给朋友就能玩了！

## ❓ 常见问题

### Q: 部署失败怎么办？
A: 最常见原因是忘记设置 Root Directory 为 `frontend`，重新部署时记得设置。

### Q: 链接太长怎么办？
A: 用短链接服务（如 suo.im）生成短链接。

### Q: 可以改域名吗？
A: 可以！在Vercel项目设置里可以改项目名，或者绑定自己的域名。

### Q: 免费吗？
A: 完全免费！Vercel免费版每月100GB流量，个人使用绰绰有余。

### Q: 如何更新内容？
A: 修改代码后，推送到GitHub，Vercel会自动重新部署。
